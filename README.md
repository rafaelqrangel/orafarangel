# @orafarangel

> **Sabedoria de milênios. Ação de 30 segundos. Mudança para sempre.**

Este é o repositório aberto da marca **@orafarangel** — o framework completo, público e versionado de como transformar filosofia clássica e espiritualidade (Bíblia, Estoicismo, Budismo, Taoísmo, Logoterapia) em **micro-ações práticas de 30 segundos**, usando a ciência comportamental de BJ Fogg.

A transparência é proposital: **o método é código aberto**. A construção desta marca é documentada em público — o GitHub vira conteúdo, os bastidores viram reels, o processo é o produto.

## A fórmula que rege tudo

**B = M + A + P** (BJ Fogg)

| Elemento | O que é | Quem entrega |
|---|---|---|
| **M** — Motivação | O princípio filosófico que dá significado | Quase todos os criadores |
| **A** — Capacidade | Ação tão simples que falhar é impossível | Poucos |
| **P** — Gatilho | Comando concreto acoplado a um hábito existente | **Só o Rafael** |

A maioria do conteúdo de desenvolvimento pessoal no Brasil entrega inspiração sem ação. Este projeto existe para fechar o gap entre **saber** e **fazer**.
