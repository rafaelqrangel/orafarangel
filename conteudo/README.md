# Conteúdo — Produção

Área de trabalho da produção de conteúdo. Aqui ficam ideias, roteiros e o calendário.

```
conteudo/
├── banco-de-ideias.md      ← ideias capturadas, uma linha cada, com pilar e status
├── roteiros/               ← um arquivo por reel/vídeo, usando o modelo
│   └── _modelo-roteiro.md  ← modelo de roteiro nos 4 tempos (copie e renomeie)
└── calendario.md           ← o que sai quando (meta: 3 reels/semana no Instagram)
```

## Fluxo de produção

1. **Capturar** — toda ideia entra no `banco-de-ideias.md` (dor + pilar, uma linha)
2. **Roteirizar** — copie `roteiros/_modelo-roteiro.md`, renomeie (`YYYY-MM-DD-slug.md`) e preencha os 4 tempos
3. **Validar** — rode o [checklist pré-publicação](../framework/template-4-tempos.md#checklist-pré-publicação)
4. **Publicar** — Instagram primeiro; depois TikTok (hook ajustado) e YouTube Shorts
5. **Registrar** — anote a performance no próprio roteiro (plays, follows, salvamentos)
